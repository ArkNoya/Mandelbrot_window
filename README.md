# Mandelbrot_window

供B站站友要求开源的Mandelbrot分型程序

使用c++&opengl编写

opengl库管理使用了glfw

>为了方便编译 下载完所有文件后 双击运行“编译.bat”

编译成功会生成一个“mandelbrot.exe”双击运行即可

# 控制键位：
字母键 M 切换渲染模式

数字键 + - 控制缩放(微量调控)

字母键 - = 控制缩放(大量调控)

数字键 上下左右 控制位置(微量调控)

字母键 WASD 控制位置(大量调控)

字母键 [ ] 控制Mandelbrot迭代次数(微量调控)

字母键 ; ' 控制Mandelbrot迭代次数(大量调控)

字母键 , . 控制Mandelbrot迭代次数(巨量调控)

字母键 H 转到一个可以一直放大的观察点
